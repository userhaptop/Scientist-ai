# Scientist-AI Prompts

这个仓库用于整理科研创新、AI-Scientist、模型创新、实验设计、论文查新相关的中文提示词。

## 目录

- `prompts/ai_scientist_research_innovation_prompt_zh.md`  
  AI-Scientist 风格科研创新总提示词，适合用于生成研究 idea、查新、反思、实验设计和 proposal 输出。

## 使用方式

复制 `prompts/` 目录中的提示词，粘贴到 ChatGPT、Claude、Qwen、DeepSeek 等大模型中，然后补充自己的研究主题、已有方法、代码背景和限制条件即可。
